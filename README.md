## Vim Configuration

Make the following folders inside `~/.vim`: `backup`, `bundle`, `colors`, `swap`, and `undo` (autoload is implicit from Pathogen). Plus, you need to install the following plugins including Pathogen:

- NerdTREE
- Fireplace
- Airline
- Surround
- Ctrl-P
- Ale
- vim-airline-themes
- Typescript
